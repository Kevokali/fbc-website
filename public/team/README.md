# Team Photos Directory

Place your team member photos in this directory with the following filenames:

## Required Images:

1. **Luke Weke** (Director & Lead Consultant)
   - Filename: `luke-weke.jpg`
   - Recommended size: 800x800 pixels (square) or 1200x800 pixels (landscape)
   - Format: JPG, PNG, or WebP

2. **CPA Calleb Masese** (Partner)
   - Filename: `calleb-masese.jpg`
   - Recommended size: 800x800 pixels (square) or 1200x800 pixels (landscape)
   - Format: JPG, PNG, or WebP

## How to Add Photos:

1. Save your photos with the exact filenames listed above
2. Place them directly in this `public/team/` folder
3. The images will automatically appear on the "Our Team" page

## Image Tips:

- Use high-quality, professional photos
- Ensure good lighting and clear backgrounds
- Square or landscape orientation works best
- Keep file sizes reasonable (under 500KB for faster loading)

## File Structure:

```
public/
  └── team/
      ├── luke-weke.jpg          ← Add Luke Weke's photo here
      ├── calleb-masese.jpg      ← Add Calleb Masese's photo here
      └── README.md              ← This file
```

Once you add the images, they will automatically display on the `/team` page!
